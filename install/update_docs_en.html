• - If your site is using a non-default theme, please update the theme to be compatible with the new version. View theme upgrade guide here (use Google Translate to see it in your language)
  • - Additional external installation modules need updating to operate on NukeViet 4.3, see update guide here (use Google Translate to see it in your language).
  • - Go to Configuration => Security Settings to set up Cross-Site and Domain Restriction functions according to your needs.
  • - Go to Configuration => Manage Modules and click on the News module name and all its virtual modules so that the system recognizes the new function. If there are multiple languages, the languages need to be processed in turn. For example: admin/index.php?language=vi&nv=modules&op=show&mod=news
  • - Go to Configuration => Manage Modules and click on the name of the Users module and all its virtual modules so that the system can recognize the new function. If there are multiple languages, the languages need to be processed in turn. For example admin/index.php?language=vi&nv=modules&op=show&mod=users
  • - If the site uses external libraries such as `phpoffice/phpspreadsheet` through composer, you need to composer update
  • - These can be done after the upgrade. You should save the two instructions above for later use.
